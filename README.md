# -
Мемо — классическая карточная игра на развитие памяти, реализованная с использованием современных веб-технологий. Игроку необходимо находить пары одинаковых карт, запоминая их расположение на игровом поле.  Этот проект демонстрирует применение принципов объектно-ориентированного программирования и чистую архитектуру кода.
